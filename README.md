# PDESoft website

This repository hosts the [PDESoft website (pdesoft.org)](https://pdesoft.org).
This small website includes an [index page](https://pdesoft.org) with links to each instance of PDESoft, and a [branding page](https://pdesoft.org/branding)
with links to high resolution versions of the logo.

The code to generate the website for [PDESoft 2024](https://2024.pdesoft.org) can be found in the [ExCALIBUR-SysGenX/pdesoft repository](https://github.com/ExCALIBUR-SysGenX/pdesoft).
